Creating and Using Templates

Create: php luna create:template email/add

Content example: <h1>{{$title}}</h1> {{map $items}} <p>{{$items.name}}</p> {{end map}}

Use: $this->view('email.add', ['title' => 'Hi', 'items' => [['name' => 'One']]]);

Single var: {{$var}}, array loop: {{map $array}} {{$array.item}} {{end map}}.
