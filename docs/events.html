Creating Events and Simple Example

To create an event, use:

php luna create:event UserRegistered

Edit app/Events/UserRegistered.php:

class UserRegistered extends Event {
    // constructor with data if needed
}

Fire event: TriggerEvent('App\Events\UserRegistered', ['user' => $user]);

Register listeners in config/events.php: 'App\Events\UserRegistered' => ['App\Listeners\SendEmail'];
