Creating Services and Simple Example

To create a service, use:

php luna create:service Article

Edit app/Services/Article.php:

class ArticleService {
    public static function getAuthorName($id) {
        $author = Authors::find($id);
        return $author ? $author->getAttribute('name') : 'Unknown';
    }
}

Use in controller: use App\Services\ArticleService; $name = ArticleService::getAuthorName(1);

Services auto-load if in app/Services; add register/boot for boot logic.
