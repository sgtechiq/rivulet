CLI Commands and Usage

php luna run [host:port] - Start server.

php luna create:model name - Create model.

php luna create:controller name - Create controller with CRUD template.

php luna create:service name - Create service with static methods.

php luna create:template name - Create HTML template.

php luna create:event name - Create event.

php luna create:rule name - Create validation rule.

php luna create:resource name - Create migration.

php luna create:seeder name - Create seeder.

php luna create:listener name - Create listener.

php luna create:job name - Create job.

php luna create -mcsterdlj name - Create multiple (m=model, c=controller, s=service, t=template, e=event, r=resource, d=seeder, l=listener, j=job).

php luna database:migrate - Run migrations.

php luna database:seed - Run seeders.

php luna database:rollback - Rollback last batch.

php luna cache:clear - Clear cache.

php luna logs:clear - Clear logs.

php luna config:clear - Clear config cache.

php luna config:cache - Cache configs.

php luna routes:list - List routes.

php luna routes:clear - Clear routes cache.

php luna routes:cache - Cache routes.

php luna queue:work [queue] [max] - Process queue.

php luna schedule:run - Run scheduled tasks.

php luna storage:link - Link storage.

php luna test:run [file] - Run tests.

php luna key:generate - Generate APP_KEY.

php luna tinker - REPL shell.

php luna optimize - Clear caches/logs.
