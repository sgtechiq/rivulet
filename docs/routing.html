Routing (Add Route, Group, Prefix, Custom File)

Add route in routes/api.php: route('GET', '/test', Controller::class, 'method');

Group: group('prefix=admin', 'middleware=auth', function () { route(...); });

Prefix: prefix('api', function () { route(...); });

Custom file: Create routes/custom.php with routes. Add to config/routes.php: 'handlers' => ['custom' => 'custom.php']; Access /custom/test.
