Validation (All Rules and Usage)

Use: validate($data, ['field' => 'required|max:10']);

Rules list and usage:

  • required: Not empty. 'field' => 'required'
  • string: Is string. 'field' => 'string'
  • bool: Is boolean. 'field' => 'bool'
  • email: Valid email. 'field' => 'email'
  • number: Numeric. 'field' => 'number'
  • integer: Integer. 'field' => 'integer'
  • url: Valid URL. 'field' => 'url'
  • date: Valid date (default Y-m-d, param:format). 'field' => 'date:Y-m-d'
  • file: Extensions (param:pdf,jpg). 'field' => 'file:pdf,jpg'
  • array: Is array. 'field' => 'array'
  • ip: Valid IP. 'field' => 'ip'
  • alpha: Letters only. 'field' => 'alpha'
  • alphanum: Letters/numbers. 'field' => 'alphanum'
  • regex: Pattern match (param:/pattern/). 'field' => 'regex:/^[a-z]+$/'
  • filesize: Max bytes (param:1024). 'field' => 'filesize:1024'
  • max: Max value/length/count (param:10). 'field' => 'max:10'
  • min: Min value/length/count (param:5). 'field' => 'min:5'
  • between: Between min,max (param:5,10). 'field' => 'between:5,10'

Throws JSON errors on fail. Custom rules in app/Rules.
