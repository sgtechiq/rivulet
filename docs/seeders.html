Creating Seeders and Adding Data

To create a seeder, use:

php luna create:seeder User

Edit the run() method in database/Seeders/UserSeeder.php:

public function run() {
    Users::create(['name' => 'Admin', 'email' => 'admin@example.com', 'password' => PassEncrypt('admin')]);
    // Bulk
    Users::query()->insert([
        ['name' => 'User1', 'email' => 'user1@example.com'],
        ['name' => 'User2', 'email' => 'user2@example.com']
    ]);
}

Run seeders: php luna database:seed
