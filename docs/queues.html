Creating Job Queues and Scheduling

Create job: php luna create:job SendEmail

Edit: public function handle() { // logic with $this->data }

Push: app()->make('queue')->push('App\Jobs\SendEmail', ['data' => 'test']);

Process: php luna queue:work [queue] [max]

Schedule: In config/schedule.php ['everyMinute' => ['job' => 'App\Jobs\Cleanup', 'data' => []]];

Cron: * * * * * php luna schedule:run

Config QUEUE_CONNECTION=db/redis for driver.
