Creating Rules like Number

To create a rule, use:

php luna create:rule Number

Edit app/Rules/Number.php:

class Number {
    public function passes($field, $value) {
        return is_numeric($value);
    }
    public function message($field) {
        return "{$field} must be a number";
    }
}

Use: validate($data, ['age' => 'number']);

Custom rules resolved automatically.
